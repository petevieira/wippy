# WIPPY
Wheeled Inverted Pendulum robot driven by ROS2

## Launch Rviz
ros2 launch urdfs display.launch.py

## Mobility Node


### State

| Variable | Name | Units |
|:---------|:----:|------:|
| $$r$$ | Cart Position |
| $$\dot{r}$$ | Cart velocity |
| $$\theta$$ | Pendulum angle |
| $$\dot{\theta}$$ | Pendulum angular velocity |
